C++ ptrdiff_t

一、定义

ptrdiff_t 是 C/C++ 标准库中定义的一个与机器相关的数据类型。ptrdiff_t 类型变量通常用来保存两个指针减法操作的结果。ptrdiff_t 定义在 stddef.h（cstddef）这个文件内。ptrdiff_t 通常被定义为 long int 类型。

二、ptrdiff_t 与 size_t

与 size_t 的区别：

类型 符号 用途 示例
ptrdiff_t 有符号 指针差值、迭代器距离 p2 - p1
size_t 无符号 大小、容量、索引 sizeof(arr)
